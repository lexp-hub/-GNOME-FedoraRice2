GNOME-FedoraRice2 - Dotfiles & Configuration
Installation
Bash git clone https://github.com/lexp-hub/-GNOME-BetterRice.git

GNOME Extensions
Blur my Shell: Essential for achieving the frosted glass effect in Kitty.
Dash to Dock: Configured with rounded corners for a "pill" look.
Desktop Icons NG (DING): Manages desktop icons.
Just Perfection: Used to clean up the UI and hide unnecessary elements.
Logo Menu: Adds the Fedora icon to the top panel.
Open Bar: Customizes the top bar with transparency and borders.
System Monitor: Real-time hardware usage in the top bar.
Tiling Shell: Adds tiling window management capabilities to GNOME.
Top Bar Organizer: Reorders top bar icons.
User Themes: Enables custom shell themes.
AppIndicator Support: Legacy support for tray icons.

Apps & Tools
Kitty: Main terminal emulator with high-fidelity image rendering.
Fastfetch: Custom JSON config with pill-shaped modules and a large Fedora logo.
Starship: A fast, cross-shell prompt engine for the "pill" command line.
CAVA: Audio visualizer for the terminal.

Fonts
Adwaita Mono Nerd Font: Required for rendering all icons and "pill" symbols correctly.

Themes & Styles
Catppuccin: GTK and Shell themes.
Kitty Theme: Custom Catppuccin-based theme with 0.85 opacity.
Icon Pack: GreyStone.
